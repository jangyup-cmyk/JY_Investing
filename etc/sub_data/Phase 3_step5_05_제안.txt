## 수정 후 `analysis_job_factory` 최종 형태

바로 교체하려면 이 부분만 이렇게 바꾸시면 됩니다.

```python
def _enum_value(value):
    return value.value if hasattr(value, "value") else value


@pytest_asyncio.fixture
async def analysis_job_factory(
    async_session: AsyncSession,
) -> Callable[..., AsyncIterator[AnalysisJob]]:
    async def _factory(
        *,
        job_type: JobType = JobType.NORMALIZE,
        target_type: JobTargetType = JobTargetType.MESSAGE,
        target_id: int = 1,
        status: JobStatus = JobStatus.PENDING,
        priority: int = 100,
        attempt_count: int = 0,
        max_attempts: int = 3,
        scheduled_at: datetime | None = None,
        started_at: datetime | None = None,
        finished_at: datetime | None = None,
        worker_name: str | None = None,
        lease_expires_at: datetime | None = None,
        last_heartbeat_at: datetime | None = None,
        idempotency_key: str = "job-key-1",
        payload_json: dict | None = None,
        error_code: str | None = None,
        error_message: str | None = None,
    ) -> AnalysisJob:
        item = AnalysisJob(
            job_type=_enum_value(job_type),
            target_type=_enum_value(target_type),
            target_id=target_id,
            status=_enum_value(status),
            priority=priority,
            attempt_count=attempt_count,
            max_attempts=max_attempts,
            scheduled_at=scheduled_at or aware_dt("2026-03-22T09:00:00+00:00"),
            started_at=started_at,
            finished_at=finished_at,
            worker_name=worker_name,
            lease_expires_at=lease_expires_at,
            last_heartbeat_at=last_heartbeat_at,
            idempotency_key=idempotency_key,
            payload_json=payload_json or {},
            error_code=error_code,
            error_message=error_message,
        )
        async_session.add(item)
        await async_session.commit()
        await async_session.refresh(item)
        return item

    return _factory
```

## 같이 보면 좋은 포인트

이 보정을 넣으면 아래 테스트들이 더 안정적으로 맞습니다.

* `test_analysis_job_service.py`
* `test_scheduler_worker.py`
* `test_preprocess_worker.py`
* `test_inference_worker.py`

이유는 실제 모델의 `AnalysisJob.job_type`, `target_type`, `status`가 enum 컬럼이 아니라 문자열 컬럼이기 때문입니다.
그래서 테스트 fixture에서 Enum 객체를 그대로 넣으면, 비교할 때 `.value`와 섞여 mismatch가 날 수 있습니다.
