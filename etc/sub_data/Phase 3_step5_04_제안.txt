좋습니다. 현재 코드베이스는 이미 `run_collector.py`가 수집 전용 진입점으로 분리되어 있고, `session.py`에 `AsyncSessionLocal()`과 `session_scope()`가 준비돼 있으며, `startup_tasks.py`는 `persist_func` 주입 방식으로 백필을 돌리도록 짜여 있습니다. 그래서 `run_worker.py`는 **수집기와 분리된 분석 워커 실행기**로 두는 것이 맞고, `scheduler_worker.py`는 **stuck job 복구**부터 시작하는 최소 구현이 가장 자연스럽습니다.   

아래 두 파일을 바로 추가하면 됩니다.

## `src/workers/scheduler_worker.py`

이 워커는 우선 `RUNNING`인데 lease가 만료된 job을 다시 `PENDING`으로 되돌리는 역할만 맡기면 충분합니다. 현재 `AnalysisJob` 모델에 `status`, `lease_expires_at`, `last_heartbeat_at`, `attempt_count`가 이미 있어서 최소 구현이 가능합니다. 

```python
from __future__ import annotations

from datetime import datetime, timezone

from sqlalchemy import select, update

from src.common.logger import logger
from src.infra.db.models import AnalysisJob, JobStatus
from src.infra.db.session import AsyncSessionLocal
from src.workers.base_worker import BaseWorker

UTC = timezone.utc


class SchedulerWorker(BaseWorker):
    async def run_once(self) -> bool:
        recovered = await self._recover_stuck_jobs()
        return recovered > 0

    async def _recover_stuck_jobs(self) -> int:
        now = datetime.now(UTC)

        async with AsyncSessionLocal() as session:
            jobs = list(
                await session.scalars(
                    select(AnalysisJob).where(
                        AnalysisJob.status == JobStatus.RUNNING.value,
                        AnalysisJob.lease_expires_at.is_not(None),
                        AnalysisJob.lease_expires_at <= now,
                    )
                )
            )

            if not jobs:
                return 0

            ids = [job.id for job in jobs]

            await session.execute(
                update(AnalysisJob)
                .where(AnalysisJob.id.in_(ids))
                .values(
                    status=JobStatus.PENDING.value,
                    lease_expires_at=None,
                    worker_name=None,
                    last_heartbeat_at=now,
                    error_code=None,
                    # 필요하면 여기서 error_message는 유지해도 됨
                )
            )
            await session.commit()

            logger.warning(
                "[%s] recovered stuck jobs: %s",
                self.worker_name,
                ids,
            )
            return len(ids)
```

이 구현은 단순하지만 실용적입니다. 지금 구조가 `claim_next_job()`에서 lease 기반으로 job을 다시 잡도록 설계되어 있으므로, scheduler는 만료된 `RUNNING` job을 되돌려주기만 해도 전체 복구성이 크게 좋아집니다. `startup_tasks.py`가 이미 백필 중 네트워크 I/O와 짧은 DB 세션을 분리하고 있으니, 분석 워커 쪽도 이런 관리용 보조 워커를 두는 편이 전체 구조와 잘 맞습니다.  

## `run_worker.py`

현재는 `preprocess`, `inference`, `scheduler` 세 종류만 지원하게 두는 것이 좋습니다. `run_collector.py`는 계속 분리해야 합니다. 이미 README도 웹 서버와 수집기를 독립적으로 띄우는 구조로 설명하고 있고, 업로드된 코드도 `CollectorWorker.start()`와 `run_startup_tasks()`를 수집 프로세스 안에서 묶고 있습니다.  

```python
import argparse
import asyncio

from src.workers.inference_worker import InferenceWorker
from src.workers.preprocess_worker import PreprocessWorker
from src.workers.scheduler_worker import SchedulerWorker


def build_parser() -> argparse.ArgumentParser:
    parser = argparse.ArgumentParser(description="Background worker runner")
    parser.add_argument(
        "--type",
        required=True,
        choices=["preprocess", "inference", "scheduler"],
        help="Worker type to run",
    )
    parser.add_argument(
        "--worker-name",
        default=None,
        help="Logical worker name",
    )
    parser.add_argument(
        "--poll-interval",
        type=float,
        default=2.0,
        help="Polling interval seconds",
    )
    parser.add_argument(
        "--job-types",
        nargs="*",
        default=None,
        help="Optional inference job type filter",
    )
    return parser


async def async_main(args) -> None:
    worker_name = args.worker_name or f"{args.type}-worker-1"

    if args.type == "preprocess":
        worker = PreprocessWorker(
            worker_name=worker_name,
            poll_interval=args.poll_interval,
        )
    elif args.type == "inference":
        worker = InferenceWorker(
            worker_name=worker_name,
            poll_interval=args.poll_interval,
            job_types=args.job_types,
        )
    elif args.type == "scheduler":
        worker = SchedulerWorker(
            worker_name=worker_name,
            poll_interval=args.poll_interval,
        )
    else:
        raise ValueError(f"Unsupported worker type: {args.type}")

    await worker.run_forever()


def main() -> None:
    parser = build_parser()
    args = parser.parse_args()
    asyncio.run(async_main(args))


if __name__ == "__main__":
    main()
```

실행은 아래처럼 하면 됩니다.

```bash
python run_worker.py --type preprocess --worker-name preprocess-1
python run_worker.py --type inference --worker-name inference-1
python run_worker.py --type scheduler --worker-name scheduler-1 --poll-interval 10
```

`inference`만 특정 job 타입으로 제한하고 싶으면 이렇게 쓸 수 있습니다.

```bash
python run_worker.py --type inference --job-types SINGLE_MESSAGE_SUMMARY
```

현재 `InferenceWorker`가 `JobType(item)`으로 문자열을 Enum으로 바꾸는 구조라면, `SINGLE_MESSAGE_SUMMARY`처럼 **실제 Enum value와 같은 대문자 문자열**을 넘기는 것이 안전합니다. 실제 `JobType` 모델 값도 전부 대문자입니다. 

적용 순서는 이렇게 추천합니다.

## `tests/workers/test_scheduler_worker.py`

이 파일은 `SchedulerWorker`가

* 처리할 stuck job이 없으면 `False`를 반환하는지
* lease 만료된 `RUNNING` job을 `PENDING`으로 복구하는지
* 아직 lease가 남아 있는 `RUNNING` job이나 애초에 `PENDING` job은 건드리지 않는지

를 검증합니다.

```python
# tests/workers/test_scheduler_worker.py

from __future__ import annotations

from datetime import datetime, timedelta, timezone

import pytest
from sqlalchemy import select
from sqlalchemy.ext.asyncio import AsyncSession

from src.infra.db.models import AnalysisJob, JobStatus, JobType
from src.workers.scheduler_worker import SchedulerWorker

UTC = timezone.utc


def now_utc() -> datetime:
    return datetime.now(UTC)


async def get_job(async_session: AsyncSession, job_id: int) -> AnalysisJob | None:
    return await async_session.get(AnalysisJob, job_id)


async def list_jobs(async_session: AsyncSession) -> list[AnalysisJob]:
    result = await async_session.scalars(
        select(AnalysisJob).order_by(AnalysisJob.id.asc())
    )
    return list(result)


@pytest.mark.asyncio
async def test_run_once_returns_false_when_no_stuck_job(async_session: AsyncSession):
    worker = SchedulerWorker(worker_name="scheduler-test", poll_interval=0.01)

    handled = await worker.run_once()

    assert handled is False


@pytest.mark.asyncio
async def test_run_once_recovers_expired_running_jobs(
    async_session: AsyncSession,
    analysis_job_factory,
):
    expired_job_1 = await analysis_job_factory(
        job_type=JobType.NORMALIZE,
        status=JobStatus.RUNNING,
        worker_name="dead-worker-1",
        lease_expires_at=now_utc() - timedelta(seconds=10),
        idempotency_key="expired-job-1",
    )
    expired_job_2 = await analysis_job_factory(
        job_type=JobType.EXTRACT_FEATURES,
        status=JobStatus.RUNNING,
        worker_name="dead-worker-2",
        lease_expires_at=now_utc() - timedelta(seconds=20),
        idempotency_key="expired-job-2",
    )

    worker = SchedulerWorker(worker_name="scheduler-test", poll_interval=0.01)
    handled = await worker.run_once()

    assert handled is True

    stored_1 = await get_job(async_session, expired_job_1.id)
    stored_2 = await get_job(async_session, expired_job_2.id)

    assert stored_1 is not None
    assert stored_1.status == JobStatus.PENDING.value
    assert stored_1.lease_expires_at is None
    assert stored_1.worker_name is None

    assert stored_2 is not None
    assert stored_2.status == JobStatus.PENDING.value
    assert stored_2.lease_expires_at is None
    assert stored_2.worker_name is None


@pytest.mark.asyncio
async def test_run_once_does_not_touch_non_expired_or_pending_jobs(
    async_session: AsyncSession,
    analysis_job_factory,
):
    non_expired_running = await analysis_job_factory(
        job_type=JobType.NORMALIZE,
        status=JobStatus.RUNNING,
        worker_name="healthy-worker",
        lease_expires_at=now_utc() + timedelta(minutes=5),
        idempotency_key="healthy-running-job",
    )
    pending_job = await analysis_job_factory(
        job_type=JobType.SINGLE_MESSAGE_SUMMARY,
        status=JobStatus.PENDING,
        worker_name=None,
        lease_expires_at=None,
        idempotency_key="pending-job",
    )

    worker = SchedulerWorker(worker_name="scheduler-test", poll_interval=0.01)
    handled = await worker.run_once()

    assert handled is False

    stored_running = await get_job(async_session, non_expired_running.id)
    stored_pending = await get_job(async_session, pending_job.id)

    assert stored_running is not None
    assert stored_running.status == JobStatus.RUNNING.value
    assert stored_running.worker_name == "healthy-worker"
    assert stored_running.lease_expires_at is not None

    assert stored_pending is not None
    assert stored_pending.status == JobStatus.PENDING.value
    assert stored_pending.worker_name is None
    assert stored_pending.lease_expires_at is None
```
