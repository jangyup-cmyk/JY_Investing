# 텔레그램 테마연동 자동매매 프로그램 인터뷰 요약 (MVP)

인터뷰가 성공적으로 마무리되었습니다! 우리가 함께 정리한 '가장 먼저 지을 집'의 설계도입니다.

## 1. 최종 요구사항 정리

### 핵심 목적
- **AI의 똑똑한 분석**과 **차트의 냉철한 데이터**를 결합하여, 사람이 없어도 24시간 자동으로 주식을 사고파는 프로그램을 만드는 것입니다.

### 주요 기능 (MVP 범위)
- **AI 친구들의 5단계 토론**: 
    - ① 노이즈 제거 → ② 전문가 3인의 관점 분석 → ③ 상호 끝장 토론 → ④ 중재자의 최종 의결 (Top 3 테마 선정)
- **100% 완전 자동 매매**: 
    - AI가 고른 테마와 13가지 차트 신호가 맞으면 자동으로 주문을 넣습니다.
- **점수제 매매 시스템**: 
    - 13가지 조건(이동평균, 거래량 등)에 점수를 매겨, 합계가 기준을 넘을 때만 안전하게 매수합니다.
- **스마트한 주문 방식**: 
    - 시장가 50% + 지정가 50%로 나누어 사고, 10분 동안 안 사지면 자동으로 취소합니다.
- **개인별 텔레그램 알림**: 
    - 나만의 봇이 나에게만 실시간으로 리포트와 매매 현황을 보내줍니다.

### 사용자 및 환경
- **사용자**: 주식 자동화를 원하는 개인 (초보자도 사용하기 쉬운 구조)
- **화면**: 우선 WEB 버전의 관리자 화면부터 만듭니다. (APP은 나중에!)
- **서버**: AWS EC2 t2.micro (무료 사양에서도 잘 돌아가게 가볍게 만듭니다.)
- **증권사**: 한국투자증권 API 활용

### 기술 스택
- **언어**: Python
- **프레임워크/라이브러리**: Flask (웹), pandas (데이터), TA-Lib (차트), APScheduler (자동화), python-telegram-bot (알림)
